# AutoFlow 🚗

Ein moderner, stylischer Gebrauchtwagen-Marktplatz – inspiriert von 12gebrauchtwagen,
aber deutlich frischer, flüssiger und schöner.

**Live-Demo:** https://haroonerika.github.io/ende/

## Features
- 🔎 Live-Filter (Marke, Preis, Kraftstoff, Getriebe, Karosserie, Kilometer, Baujahr)
- ❤️ Favoriten (im Browser gespeichert)
- ↕️ Sortierung (Preis, Kilometer, Baujahr, Empfohlen)
- 🪟 Detailansicht pro Fahrzeug (Modal)
- 🌗 Hell-/Dunkel-Modus
- 📱 Voll responsiv, flüssige Animationen
- ⚡ Reines HTML/CSS/JS – keine Abhängigkeiten, läuft offline

## Lokal starten
Einfach `index.html` im Browser öffnen.
